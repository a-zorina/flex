Eliminate contract and return all the remaining crystals to dst

Parameters
ignore_errorsIgnore error returned by depool for transferStake. finalize will not work if the contract is in "lend mode" or last_depool_error_ != 0 && !ignore_errors.
Warning
Do not use ignore_errors=true, unless you are really sure that your stake in DePool is empty or insignificant.

STATUS_DEPOOL_CLOSED / STATUS_NO_PARTICIPANT are not considered as an errors forbidding finalize
